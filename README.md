# EcoAR - Bangkok Green Scanner & Sorting Game 🌿

EcoAR is an immersive, mobile-responsive Single Page Application (SPA) designed to make recycling engaging and educational. Inspired by the **Greener Bangkok** platform's environmental initiatives, the app combines client-side AI object scanning, an active leaderboard, and an interactive sorting arcade game.

---

## ✨ Features

1. **AR Camera Scanner (`scanner.js`)**
   - Uses WebRTC camera access with a futuristic scanning HUD overlay (reticles, scanner line, real-time telemetry).
   - Integrates local **TensorFlow.js (COCO-SSD)** object detection to classify waste directly in the browser (no backend servers required).
   - Maps standard detections (bottles, cups, bananas, cell phones) into trash bin categories.
   - Includes a **Simulation Mode** and a custom photo uploader so that users on desktops or devices without cameras can still experience full scanning features.

2. **Pop-up Detail Cards & Minimalist Bins**
   - Shows what type of bin the trash belongs to: **Recyclable (Blue)**, **Organic (Green)**, **Hazardous (Red)**, or **General (Gray)**.
   - Displays matched **3D minimalist bin images** generated by AI.
   - Lists step-by-step cleaning/disposal instructions and interesting eco-facts.

3. **Eco-Sort Challenge Arcade Game (`game.js`)**
   - A fast-paced drag-and-drop game designed on HTML5 Canvas.
   - Emojis representing trash fall from the sky; players drag/swipe them into correct bins.
   - Renders a cartoon landscape featuring grass hills, clouds, and a sun.
   - Features touch swiping (for mobile compatibility), combo multipliers, particle sparkles, and floating score texts.
   - Awards bonus XP directly to the player's global profile.

4. **Simulated Active Leaderboard (`leaderboard.js`)**
   - Displays ranks, eco-badge levels, and item stats.
   - Competitors are themed as Bangkok citizens (e.g., *Somchai_GreenBKK*, *Pim_EcoBangkok*).
   - Ranks update in real-time as a background simulation mimics other active players scanning waste.
   - Stats are persisted in the browser's `localStorage` across page updates.

5. **Greener Bangkok Aesthetic**
   - Styled after `https://greener.bangkok.go.th/` using forest green accents (`#00744B`), clean light background themes, neobrutalist borders, and the **Chakra Petch** Google Font.
   - Hovering the mouse pointer over any button or tab highlights the text/icon in Forest Green.

## 🚀 How to Run Locally

Since this is a client-side static application, you can run it using any simple local server.

### Using Python (Installed on this system)
1. Open terminal inside this folder.
2. Start the HTTP server:
   ```bash
   python -m http.server 8000
   ```
3. Open your browser and go to: **[http://localhost:8000](http://localhost:8000)**

*(Note: To test the AR camera on a mobile phone, connect your phone and computer to the same Wi-Fi network and access your computer's local IP address, e.g., `http://192.168.1.XX:8000`).*

## 📦 How to Put on GitHub (Web Upload Method)

If Git is not installed on your computer, you can upload this project directly via your browser:

1. Log in to [GitHub](https://github.com/) and click **New** to create a repository.
2. Give it a name (e.g., `eco-ar-bangkok`) and click **Create repository**.
3. Under the **Quick setup** section, click the link that says **"uploading an existing file"**.
4. Drag and drop these files and folders from your computer:
   - `index.html`
   - `styles.css`
   - `app.js`
   - `scanner.js`
   - `game.js`
   - `leaderboard.js`
   - `README.md`
   - `assets/` (folder containing the 4 bin images)
5. Click **Commit changes** at the bottom.

## 🌐 Deploy to GitHub Pages (Free Hosting)

Once your code is on GitHub, you can publish it so anyone can open it on their phone:

1. Inside your GitHub repository, go to the **Settings** tab.
2. Scroll down the left sidebar and click **Pages**.
3. Under **Build and deployment**, select **Deploy from a branch**.
4. Choose `main` (or `master`) branch and select `/ (root)` folder.
5. Click **Save**.
6. Wait 1–2 minutes, and GitHub will provide a live URL (e.g., `https://username.github.io/eco-ar-bangkok/`) that you can open on any mobile phone camera!
